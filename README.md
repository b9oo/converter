# MP4 to MakeCode Arcade Song Converter

Upload an MP4 file and extract audio for conversion into MakeCode Arcade songs.

## Deploy

1. Create a GitHub repository.
2. Upload all files.
3. Go to Settings → Pages.
4. Enable GitHub Pages from the main branch.

## Future Features

* Pitch detection
* Automatic note recognition
* MakeCode song export
* TypeScript extension generation
